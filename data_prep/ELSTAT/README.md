# ELSTAT Data (Tourism Turnover)

Get original data from: https://www.statistics.gr/el/statistics/-/publication/SBR03/-

In 2025, the files were the following:
```
A1201_SBR03_TB_QQ_01_2019_02_2025_01_F_GR.xlsx
A1201_SBR03_TB_QQ_01_2019_02_2025_02_F_GR.xlsx
```

Rename them to:
```
SBR03_01.xlsx
SBR03_02.xlsx
```
respectively. Pray the formatting has stayed the same. Otherwise the
`data_ELSTAT_preprocess.py` script will need some adjustments.
